# レビュー

## チェックリスト
* 誤字脱字が無いこと
* 曖昧な表現を使用していないこと
* 読みやすい構成になっていること
* 自然な日本語の表現になっていること
* マークダウン記法が正しく使用されていること
* 以下の書き方ルールに沿っていること。なお、製品名やサービス名などの固有名詞は、例外とすること

1. カタカナが続いた場合は、英単語ごとに半角を追加すること
例) コンピュータサイエンス -> コンピュータ サイエンス

2. 英語と日本語が混在する場合は、英単語の前後に半角スペースを追加すること
例) APIを使用する -> API を使用する

3. 英単語と英単語の間は半角スペースを追加すること
例) MachineLearning -> Machine Learning

4. 英単語と数字の間は半角スペースを追加すること
例) Windows10 -> Windows 10

5. 半角記号の後ろには半角スペースを追加すること
例) 対象:Windows 10 -> 対象: Windows 10

## 出力
上記のチェックリストに基づいてレビューを行い、指摘事項と改善案をチャットで共有し、改善案を反映した Markdown ファイルを新規に作成してください。(ファイル名:<対象ドキュメント名>_reviewed.md)
なお、出力ディレクトリについてはレビュー対象の同一ディレクトリに出力してください。
