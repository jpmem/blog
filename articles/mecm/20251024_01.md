# WSUS の脆弱性(CVE-2025-59287)への対応
みなさま、こんにちは。 Configuration Manager サポート チームです。  
「CVE-2025-59287 Windows Server Update Service (WSUS) のリモートでコードが実行される脆弱性」の対処についてご案内いたします。

WSUS の脆弱性の詳細に関しては [CVE-2025-59287](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2025-59287) でご説明しておりますが、
弊社ではこの問題に対する更新プログラムを 2025/10/23(米国時間) で 緊急リリースしております。

ご迷惑をおかけし申し訳ございませんが、お使いの Windows Server のバージョンをご確認いただき、以下の中から該当する帯域外更新プログラムをインストールいただきますようお願いいたします。

- Windows Server 2016 [KB5070882](https://support.microsoft.com/ja-jp/topic/october-23-2025-kb5070882-os-build-14393-8524-out-of-band-3400c459-db78-48bc-ae69-f61bff15ea7c)
- Windows Server 2019 [KB5070883](https://support.microsoft.com/ja-jp/topic/october-23-2025-kb5070883-os-build-17763-7922-out-of-band-860bc03c-52fb-407c-89b2-14ecf4893c5c)
- Windows Server 2022 [KB5070884](https://support.microsoft.com/ja-jp/topic/october-23-2025-kb5070884-os-build-20348-4297-out-of-band-9c001fdc-f0d2-4636-87bb-494a59da55d0)
- Windows Server 2025 [KB5070881](https://support.microsoft.com/ja-jp/topic/october-23-2025-kb5070881-os-build-26100-6905-out-of-band-8e7ac742-6785-4677-87e4-b73dd8ac0122)
 
なお、ConfigMgr につきましても、ソフトウェア更新ポイントの役割を持つサーバーは WSUS を部品として利用しておりますため、上記の帯域外更新プログラムの適用対象となります。
 
なお、上記の更新プログラムのインストールを行うと、もしカタログ同期に失敗しエラーが発生した場合に、その詳細がコンソールのレポート上に反映されなくなるという別の問題が発生いたしますが、その問題に関しては引き続き修正を行っております。
カタログ同期の失敗を引き起こす問題ではございませんが、追って別の更新プログラムで修正予定(リリース時期は未定)でございますため、まずは脆弱性への対処を優先するために、上記の帯域外更新プログラムのインストールを行うことをご検討いただきますようお願いいたします。
 
また、本脆弱性については 当初 2025/10/14 にリリースされた月例更新プログラムで修正をご案内しておりましたが、修正に不足があったため、帯域外更新プログラムが再度リリースされております。
ご迷惑をおかけして大変恐縮でございますが、改めて上記帯域外更新プログラムの適用をご検討いただきますようお願いいたします。
